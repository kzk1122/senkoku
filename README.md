# 線刻(せんこく / senkoku)

Apple Pencil でイラストの基礎「運筆」を練習する Web アプリ(PWA)。
お手本の線をなぞると **精度・網羅・滑らかさ** の3軸で採点されます。70点以上で合格。課題は自由に選べます。

## 特徴

- **28の線課題**:直線(4方向)・カーブ(はらい・なみ各種)・まる(両回り・同心円・うずまき)・大カーブ(アーチ・S字・むげん∞)・筆圧(キープ・強弱・抜き)・ハッチ(ななめ・たて・クロス)。どれからでも挑戦可能
- **筆圧コントロール課題**:一定筆圧を保つ「筆圧キープ」、徐々に強める「だんだん強く」、はらいながら抜く「スッと抜く」。補助線の太さが目標筆圧のお手本になり、筆圧の一致度が採点に加わる
- **ハッチング課題**:斜めの平行線を6本、等間隔に。6本描き終えると自動採点。間隔の乱れ・線の抜けが減点対象
- **かたちの練習(Stage 2)**:図形分解による人体アタリ課題8種(頭部・胴体・全身・横顔・腕・手・脚・足)と、プロシージャル生成ポーズのタイマー式ジェスチャードローイング(30秒/60秒/2分)
- **模写の練習(Stage 3)**:モチーフ5題材(三日月・富士・なみもよう・瓢箪・うめの花)を「隣に表示 → 下敷き → 記憶」の3モードで模写。採点時にコーラルの差分オーバーレイでズレを確認できる
- **Apple Pencil 対応**:PointerEvent の `pressure` で筆圧を線の太さに反映。`getCoalescedEvents()` による高解像度サンプリング。ペン検出後はタッチ入力を無視するパームリジェクション付き
- **ランクバッジ採点**:S・A・B・C・D のステッカー風バッジで結果を表示
- **補助線の段階調整**:濃 → 淡 → 無 の3段階。慣れたら補助線なしで挑戦
- **進捗保存**:最高得点を `localStorage` に保存(将来的に IndexedDB へ移行予定)
- **オフライン対応**:Service Worker によるキャッシュで PWA としてホーム画面に追加可能

## 動かし方(ローカル)

ビルド不要の静的サイトです。Service Worker を使うため、ローカルサーバー経由で開いてください。

```bash
# 例: Python
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

iPad で試す場合は、同じ Wi-Fi 上の PC のローカル IP(例 `http://192.168.x.x:8000`)を Safari で開きます。

## 採点アルゴリズム

| 指標 | 重み | 内容 |
|---|---|---|
| 精度 | 50% | 描いた各点からお手本パスまでの平均距離(お手本サイズで正規化) |
| 網羅 | 30% | お手本の各サンプル点の近傍(サイズの5.5%以内)を通過した割合 |
| 滑らか | 20% | 前平滑化(等間隔リサンプリング → 3点移動平均×2)した線の角度変化(二階差分)から、お手本自身の曲率を差し引いた「余分なガタつき」 |

さらに網羅率によるゲートがあり、線が途中で終わっている場合(網羅率90%未満)は合計点が段階的に減点され、40%以下では0点になります。「一筆で最後まで描き切る」ことを最優先で評価します。

筆圧課題では、目標筆圧カーブとの平均誤差による「筆圧」軸が加わった4軸配分(精度35% / 網羅25% / 滑らか15% / 筆圧25%)になり、筆圧が大きく外れている場合はゲートによる減点も受けます。

合格ラインは **70点**。ランクは 90+ S / 80+ A / 70+ B / 55+ C / それ未満 D。

## ファイル構成

```
senkoku/
├── index.html            # Stage 1: 線の練習
├── keitai.html           # Stage 2: かたちの練習
├── mosha.html            # Stage 3: 模写の練習
├── css/style.css         # ポップなデザイントークン(クリーム×コーラル×ティール)
├── js/scoring.js         # 共通採点エンジン
├── js/surface.js         # canvas 入力・筆描画の共通部品
├── js/app.js             # Stage 1: 課題定義・入力処理・進捗
├── js/keitai.js          # Stage 2: アタリ課題・ポーズ生成・タイマー
├── js/mosha.js           # Stage 3: 題材・3モード・正規化採点・差分表示
├── icons/icon.svg        # PWA アイコン
├── manifest.webmanifest  # PWA マニフェスト
├── sw.js                 # Service Worker(オフラインキャッシュ)
└── README.md
```

## 技術メモ

- 筆圧はマウスだと常に `0.5` 相当。Apple Pencil では 0〜1 の実値が取れ、線の太さ `2 + 7 × pressure` px に反映
- キャンバスは `devicePixelRatio` でスケーリングし、Retina でも線がにじまない
- `touch-action: none` でスクロールとの競合を防止
